Qué se cubrió desde tu rol

  • JavaScript solo funciona en formulario: No toca API calls ni flujos sin intervención. Implicación: limitación clara para lógica omnicanal.

Tu aportación sobre seguridad

La solución: Cloud Code con un custom agent de auditoría, o Notion como segundo cerebro. Identificaste una brecha real en flujos de desarrollo asistido.
